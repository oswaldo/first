# AI Images

This folder contains AI-generated images for various uses such as:
- Tutorials
- Webpages
- Media posts
- Presentations

## Convention

Each image file (e.g., `image_name.png`) is paired with a Markdown file of the same name (e.g., `image_name.md`).

The Markdown file contains:
1.  **Context**: What was happening when the image was requested.
2.  **Prompt**: The exact text prompt used to generate the image.
